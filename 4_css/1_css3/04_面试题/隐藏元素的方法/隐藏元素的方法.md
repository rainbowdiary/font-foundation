# 隐藏元素的方法
1. opacity: 0 
  设置元素隐藏的方法：opacity：0
        特点： 1. 占据文档流，可以做用户交互，可以用来做动画效果
                  例如：鼠标悬才停显示按钮
              2. 隐藏了也可以触发事件
        注意：
          该属性是兼容IE9以上的浏览器，
          IE8 以及更早的版本支持替代的 filter 属性，
          例如： filter:Alpha(opacity=50)。
2. display: none
   1. 不占据文档流，他和他的子元素添加动画效果会失效
3. visibility: hidden
   1. 会占据文档流，对网页起到布局效果（同opacity）
   2. 不同的是不会响应用户交互
4. position: 定位到不在视图内的位置
   1. 意义： 将元素脱离文档流，移出视觉范围
   2. 添属性后既不会影响布局，又能让元素保持可以操作
