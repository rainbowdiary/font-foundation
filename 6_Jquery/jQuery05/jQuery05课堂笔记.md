# jQuery 第五天 案例

## 回顾

### 事件机制

```
绑定：
on('事件名', fn)
事件名(fn)
one('事件名', fn)

解除绑定：
off()

事件委派（委托）：
on()  父元素调用on方法，子元素选择器作为on的第二个参数

事件的自动调用：
trigger()


jQuery的事件：
mouseenter()
mouseleave()
hover()
foucsin()
focusout()

```





### 动画效果

```
基本效果：
hide()
show()
toggle()

滑动效果：
slideUp()
slideDown()
slideToggle()

淡入淡出：
fadeOut()
fadeIn()
fadeToggle()
fadeTo()


自定义动画：
animate({}, itme, fn)

动画控制：
delay()
stop()
finish()

```
